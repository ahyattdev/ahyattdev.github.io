OrangeIRC

An IRC client for iOS written with Swift 3. It supports multiply servers and multiply rooms. When on iPad, the interface allows the list of rooms and chat logs to be displayed side-by-side.

Source code

Zombie Party

A top-down arcade zombie shooter written with the HTML5/JavaScript Phaser game library. It features 10 levels of increasing difficulty, sounds, reloading, ammunition, health, lava, and water. It can be played in any modern browser.

Live version

Radio Operator

A game also written with Phaser that simulates the experience of a radio operator in the early 20th century. It features a real transcript from the sinking of the RMS Titanic in morse code that the user must transcode to the latin script. Also included is a fictional transcript from Pearl Harbor.

Source code

Live version

UNIX Permission Visualizer

A simple Swing GUI written in Java that makes creating and understanding UNIX permission octals easier. It supports the special octal (setuid, etc) in addition to the three permission octals.

Source code

TypeTab

An iPhone modification of the stock Safari app that improves it by allowing you to type text into the address bar in less taps. When you create a new tab, instead of having to tap on the address bar to type into it, you are instantly able to type into it. You can also type into the address bar immediatly after opening or resuming Safari. This small change makes a big difference for people who frequently create new tabs. As of July 2016, the download count of each version totals to over 30,000.

coreutils-swift

An experimental re-write of over 15 common UNIX commands in Swift 3. Most notable are ls, cat, mkdir, and wc.
